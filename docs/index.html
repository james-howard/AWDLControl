AWDL CONTROL

AWDL Control is a Mac app that automatically disables Apple Wireless Direct Link when the active application is a game, and then automatically brings it back up when you quit the game or switch back to your desktop. This can help prevent network lag spikes in games when you don't need AirDrop or related features.
